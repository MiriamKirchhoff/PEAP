# PEAP
Code for the publication on PEAP
# Overview

PEAP is a MATLAB-based analysis pipeline for phase extraction of EEG data. It compares commonly used approaches.

# Requirements

To run this project, the following dependencies are required:

- MATLAB (tested with 2025a)
- EEGLAB
- Circular Statistics Toolbox by Philipp Berens


# Data availability

Raw data can not be shared, but processed data required for the analyses is provided. For questions on data availability, please contact Prof. Ziemann.
